ぺおるサブ限マイクラサーバーへようこそ！
このサーバーはサブスク限定のサバイバルサーバーです
JavaEdition,BedrockEditionの両方が参加可能(クロスプレイ)です
ここでは参加方法やルール、注意事項などをまとめています
よく読み、節度を持ったプレイを心掛けましょう


参加方法

参加方法を段階的に説明していきます

1.認証

まず募集くんを使い、MinecraftIDを認証します
募集くんのDMを開き、/authを実行してください

     /auth id: < MinecraftID > type: < java edition / bedrock edition * >

* BedrockEdition（以降BE）は所謂統合版のことで、JavaEdtion以外の全て（スマホ,WIN10,PS4/5,Switchなど）のプラットフォームのことを指します

     

BEの場合、仕様上一時的にホワイトリストを無効化し、サーバー参加後に追加する必要があります
送信後、画像のようなポップが表示されるので、中に希望する日時を記入し、送信してください

     

承認されるまでに時間がかかる場合がありますが、ご了承ください

2.承認後

承認されると、画像のようなメッセージが募集くんからDMで届きます

     

承認されるとサブ限マイクラチャンネルが開放されます
サーバーipなどは同チャンネルのサーバーステータススレッドにて確認できます
ipが変更された場合はminecrafterロールにメンションを飛ばします

     

サーバーステータススレッドに記載されているIP、PORTを入力し参加できます

BEの場合は記載した日時にホワリスを無効化するので、その際に参加してください
対応できなかった場合や、問題が発生した場合は ふろ からDMを送りますので、プライバシー設定の確認なども併せてお願いします


現在、サーバーを中継するサービスである SecureShareNet のAプランを使用しています
このサービスでは、アドレスが定期的(最大5日)に変更されるため、都度/ipを使用して確認する必要があります
参加できなかった場合は、ipが更新されているか確認してください
確認できない/更新されていなかった場合はサーバーを落としている可能性があります

アドレス固定化が付くCプラン分の費用(500円/月)を代わりに払ってくれる石油王が居たら連絡待ってます

コンシューマー(PS4/5,Switch)の場合

コンシューマーで外部サーバーに接続するには操作をする必要があります
この操作には2つ方法があります
一つはDNS構成を変更する方法
もう一つはフレンドを追加する方法です
揚げたてのポテトさんのブログ で詳しいやり方が解説されているので、そちらを参照してみてください

ルール

サーバーに参加するにあたってのルールは1つです

1. サブ限Discordサーバーでのルールを遵守すること

上記に反した場合は一時的或いは無期限にbanする可能性があります

注意事項

ルールに通ずるものもありますが、サーバー参加後の注意事項を幾つか記載します

1. スポーンプロテクション

これはサーバースポーン地点を保護するサーバーデフォルトの仕様です
参加すると ダイヤブロック で囲まれたところにスポーンすると思います
この ダイヤブロック に囲まれた範囲(+1)では、上下共に破壊及び設置が一切出来ません
また、設置を行った際ブロックは消費してしまうので気を付けて下さい

2. サーバー／チャットログ

サーバーデフォルトの仕様として、サーバー及びチャットログはすべて保存されています
場合によりそのログを確認、使用することがあります

3. CoreProtect

本サーバーでは、CoreProtectというプラグインを導入しています
このプラグインでは、対象の操作(ブロック破壊や設置を含む)の内容や時間など詳細なログを記録、参照し巻き戻すことができます
上記同様場合によってはそのログを確認、使用することがあります

4. トラブル・不具合時の対応

メンバー同士でのトラブルや不具合があった場合はすぐに ふろ にDMを送ってください
報告する際は起こった時間や場所なども添えていただけるとありがたいです
また、誤ってブロックを破壊してしまった、TNTを着火させてしまったなどの場合も報告していただければ対応します

5. 再認証

本サーバーでは1日毎にサブスクしているかどうかを判定し、サブスクされていなかった場合はロールの剥奪とホワイトリストからの削除を行っています
削除されてしまった場合は再度/authを実行して再認証してください

6. BEのテクスチャ不具合

本サーバーはGeyserを使用し、BEの通信パケットをJEの通信パケットに変換して*参加させている状態なので、BEではテクスチャ表示などに不具合が生じる場合があります
その場合はGeyserMCが公式で出している GeyserOptionalPack というリソースパックを適用してみてください
上記リンクからGeyserOptionalPackのInformationを選択し、DownloadGeyserOptionalPackを選択すると.mcpackファイルをダウンロードできます

* Minecraft公式から認められています

その他不明点などありましたら ふろ にDMを送ってください

ルールを守って楽しいサーバーを作ろう
